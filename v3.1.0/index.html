PauliOperators

Documentation for PauliOperators, a Julia package for efficient manipulation of Pauli operators and quantum states using symplectic bitstring representations, supporting up to 128 qubits.

Where to look

The README is the user manual: it covers installation, every type, and every user-facing feature with examples. Start there.

This site covers what the README doesn't — the lower-level material:

  • Pauli Representation — the symplectic $(z|x)$ encoding, the exact phase conventions (symplectic_phase, coeff, why Y prints as y), bit ordering, and the bit-level product/commutator identities the kernels are built on.
  • Data Structures & Performance — how the Dict-backed PauliSum and the flat, zero-allocation SparsePauliVector are laid out, how the windowed evolution engine works, and how to choose between them.
  • Truncation — the weight measures, the exact semantics of every truncation strategy, error tracking with correction accumulators, and how to extend the system.
  • Migration Guides — updating downstream packages to PauliOperators v3.
  • Reference — docstrings for all types and functions.
